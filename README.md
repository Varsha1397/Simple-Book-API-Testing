# Simple Book API Testing Project

## 📖 Project Overview

This project demonstrates **API Testing** performed using the **Simple Book API**. The objective was to validate REST API endpoints, verify request and response behavior, and ensure correct order processing functionality.

---

## 🎯 Testing Objectives

* Validate API endpoints functionality
* Verify authentication workflow
* Test order creation and retrieval
* Validate response structure and status codes
* Perform API automation using Postman scripts

---

## 🔗 API Under Test

Simple Book API
